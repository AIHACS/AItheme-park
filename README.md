# AIテーマパーク

AI初学者、未経験者がURLを開くだけで遊べる、プロンプト型のAI体験サイトです。

## 使い方

1. `index.html` をブラウザで開く
2. 気になるAIアトラクションの「遊ぶ」を押す
3. 「コピーしてChatGPTで開く」を押す
4. ChatGPTに貼り付けて体験を始める

## GitHub Pagesで公開する場合

1. このフォルダの中身をGitHubリポジトリに置く
2. GitHubの `Settings` から `Pages` を開く
3. `Deploy from a branch` を選ぶ
4. `main` ブランチと `/root` を選ぶ
5. 表示されたURLを共有する

APIキーやサーバーは不要です。
