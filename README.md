# AIテーマパーク

AI初学者、未経験者がURLを開くだけで遊べる、AIへの指示文型のAI体験サイトです。

## どんな技術を使ったのか

- HTML / CSS / JavaScript だけで作成
- github siteを使って静的サイトホスティング
- 5問の診断ロジックを JavaScript で実装
- 診断結果に合わせて、おすすめAIとAI活用法10選を表示
- AIアトラクションのAIへの指示文をコピーして、ChatGPTで使える仕組み
- GitHub Pagesなどの静的ホスティングで公開可能

## AIをどう使ったのか

- AI初心者でも使いやすいように、専門用語を減らした文章へ調整
- 勉強、レポート、プレゼン、就活、料理、旅行など、目的別のAI活用例を整理
- 各アトラクションごとに、ChatGPTへ貼り付けるだけで体験を始められるAIへの指示文を作成
- 診断の回答内容から、ユーザーに合うAIアトラクションをおすすめする流れを設計

## ユーザーストーリー

- AIを使ったことがない人として、最初に何を聞けばいいかわからないので、診断から自分に合うAIを見つけたい
- 大学生として、レポートや発表の準備を楽にしたいので、勉強AIやレポートAIをすぐ使える形で開きたい
- 日常生活でAIを試したい人として、料理、旅行、家計管理など身近なテーマからAI活用を体験したい
- 創作に興味がある人として、画像、音楽、SNS投稿、アプリ制作などのアイデア出しにAIを使いたい
- AIに詳しくない人として、難しい設定なしでAIへの指示文をコピーし、ChatGPTに貼るだけで体験を始めたい

## 使い方

1. https://aihacs.github.io/AItheme-park/ をブラウザで開く
2. 5問のAIお試し診断に答える
3. 診断結果でおすすめAIを確認する
4. 「テーマパークへ行く」を押す
5. 気になるAIアトラクションの「遊ぶ」を押す
6. 「コピーしてChatGPTで開く」を押す
7. ChatGPTに貼り付けて体験を始める
